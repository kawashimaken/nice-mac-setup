# Docker Setup
Since you have already had vagrant, How about give coreos a spin?

[https://github.com/coreos/coreos-vagrant](https://github.com/coreos/coreos-vagrant)

```
git clone https://github.com/coreos/coreos-vagrant/
cd coreos-vagrant
vagrant up
vagrant ssh
```
